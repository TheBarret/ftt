# ftt

![](https://i.imgur.com/NoEGBpo.png)


*Packages required*
- RTL_433
- Screen
- inotifywait
- Tail
- PHP (Tested on 8.0)

*Settings*

Change the webhook URL in the `config.inc.php` file

*Starting & Stopping*

Starting the application

`johndoe@linux:~/ftt$./start`

Stopping the application

`johndoe@linux:~/ftt$./stop`
